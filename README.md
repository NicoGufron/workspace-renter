
This is a **NextJS** project, built with **TailwindCSS**, **FramerMotion** and **dndKit** to handle drag and drop, and last but not least, **Zustand** as its global state management. Built to complete coding challenge for desent.io. 

It is truly a remarkable challenge, I learned new stuffs based from this challenge only. So, thank you Desent!

## Features
Features available in **Workspace Renter**

- Drag and drop freely items into the Workspace Preview
- Uses Supabase
- Able to add or remove items
- Uses calendar to increase or decrease the price of rent based on availability days
- Able to use protection fee (10% increase fee)
- Uses search bar to find items
- Uses catalog to help better list items

I have some ideas to improve the project, here are some I've thought about:

- Able to make presets (gaming, coding, or office work)
- AI recommendation
- Comparison (should be able to compare between basic vs regular vs pro setups)
  

## Getting Started

  

First, run the development server:

  

```bash

npm  run  dev

# or

yarn  dev

# or

pnpm  dev

# or

bun  dev

```

  

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

  

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

  

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

  


  

You can check out [the Next.js GitHub repository](https://github.com/vercel/next.js) - your feedback and contributions are welcome!
